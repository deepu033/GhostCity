## 📌 About the Project
GhostCity is a safety-focused navigation platform designed to help women and students find the **safest routes during night travel**.  

It analyzes multiple safety factors like **location data, crowd density, and user reports** to recommend secure paths.

---

## ✨ Key Features
- 🛣️ **Safe Route Recommendation** – Suggests safer paths instead of shortest ones  
- 📍 **Location-Based Analysis** – Evaluates routes using real-time data  
- 🚨 **Emergency Alerts** – Quick alert system for unsafe situations  
- 👥 **Community Reports** – Users can report unsafe areas  
- 📱 **Responsive UI** – Works smoothly on all devices

## 🧠 How It Works
1. User enters source & destination  
2. System analyzes multiple routes  
3. Safety score is calculated  
4. Safest route is recommended  
5. Real-time tracking ensures safety
